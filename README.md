# Educational Companion System

An AI-powered educational companion designed to enhance learning through
personalized recommendations, intelligent time management, and gamification.

## Features
- Personalized learning recommendations (Content-Based, Collaborative, Hybrid)
- Intelligent task scheduling and time management
- Gamification system (XP, levels, badges)
- Explainable AI recommendations

## Tech Stack
- **Backend:** ASP.NET Core, Entity Framework Core, PostgreSQL
- **AI Service:** FastAPI (Python), scikit-learn
- **Frontend:** React, TypeScript
- **Infrastructure:** Docker, Redis

## Architecture
Monorepo structure containing:
- Backend API
- AI microservice
- Frontend application
- Infrastructure and documentation
